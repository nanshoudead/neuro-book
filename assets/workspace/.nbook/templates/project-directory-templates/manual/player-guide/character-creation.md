# 角色创建

本文件用于说明如何创建当前 Project 的起始角色。它应该只写创建角色所需的信息，不提前暴露隐藏真相。

## 创建前可见信息

列出玩家在创建角色前可以知道的世界常识，例如：

- 世界基调。
- 起始地区。
- 常见身份。
- 初始危险。
- 可公开的规则前提。

需要稳定来源时，链接到 `lorebook/...`。

## 创建步骤

按当前 Project 需要填写：

1. 姓名或称呼。
2. 外貌与年龄感。
3. 出身或来处。
4. 性格倾向。
5. 初始物品。
6. 已知关系。
7. 初始能力、资源和限制。
8. 体验偏好。

## 默认值

说明哪些项目可以直接使用默认值。预设角色见 `manual/player-guide/playable-characters/README.md`。

## 不应提前确认

列出创建阶段不应直接决定的内容，例如隐藏身份、幕后真相、未来剧情、敌我最终立场或绕过代价的永久胜利条件。

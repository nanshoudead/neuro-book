# 主持人速查屏

本文件是主持人速查屏。它不替代 `rules-guide.md` 或 `lorebook/`，只把跑团时最常用的入口和判断放在一起。

## 常用入口

- 开局：`manual/README.md`
- 玩家手册：`manual/player-guide/README.md`
- 角色创建：`manual/player-guide/character-creation.md`
- 预设可玩角色：`manual/player-guide/playable-characters/README.md`
- 世界导览：`manual/world-guide.md`
- 规则：`manual/rules-guide.md`
- 主持：`manual/gm-guide.md`

## Tick 速查

| 步骤 | 检查 |
|---|---|
| 处境 | 玩家角色现在能感知什么？ |
| 行动 | 玩家正在尝试什么？ |
| 回应 | 是否需要裁决、消耗或代价？ |
| 后果 | 世界发生了什么变化？ |
| 选择点 | 下一步有哪些明显方向？ |

## 设定原典入口

在这里列出常用 `lorebook/...` 入口，例如世界主设定、主要势力、重要地点、核心系统和创作边界。

## 裁决速查

- 玩家输入是尝试，不是已确认事实。
- 行动先判断：是否可行、是否有风险、需要什么资源、失败代价是什么。
- 玩家可见信息来自 `manual/player-guide/`、角色可知内容和当前场景。
- 隐藏真相用线索呈现，不直接剧透。

## 扩展入口

记录本 Project 的随机表、遭遇表、战斗速查、价格表或其他跑团辅助材料入口。若未来由脚本生成，也只在这里放人类可读的索引和说明。

# RP Leader Project Context

本文件只记录当前 Project 对 `rp.leader` 的补充上下文、优先读取路径和接手说明。通用 RP 目录和 simulation 合同由 reference 文档承担，不在这里复制。

## Role Boundary

- `rp.leader` 负责 RP 统筹、用户交流、开局引导、陪伴式互动、桌规解释和调用模拟内核。
- `simulator.leader` 负责世界模拟、裁决、subject / entity state 和 simulation runs。
- `rp.leader` 不应把 meta 讨论自动写成世界事实。需要落地时先确认目标目录和权限。

## Reading Hints

- 开始 RP 前先读取 `manual/README.md`、`manual/player-guide/README.md`、`manual/player-guide/character-creation.md`、`manual/player-guide/playable-characters/player.md` 和 `manual/gm-guide.md`。
- 需要规则时读取 `manual/rules-guide.md` 和 `manual/reference.md`。
- 需要稳定世界事实时读取相关 `lorebook/` 条目，而不是复制整本 lorebook。
- 需要裁决世界变化时创建或复用 `simulator.leader`。

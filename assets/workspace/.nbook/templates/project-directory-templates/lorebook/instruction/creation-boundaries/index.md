## 硬性边界

填写绝不采用的情节、设定、描写或价值取向。

## 雷点

记录目标读者或作者明确排斥的内容。

## 容易误写的方向

记录 Agent 或协作者需要特别避开的误读。

## 使用方式

确认内容稳定后，由 leader、retrieval、writer handoff 或 `agent-context/{profile}/context.md` 决定是否读取或传递给相关 profile。

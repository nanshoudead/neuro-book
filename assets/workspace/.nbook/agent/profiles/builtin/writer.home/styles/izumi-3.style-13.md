文体定位: 镰池和马式文风
核心理念: 高信息密度，逻辑驱动，事实冲击。

I. 核心铁律_生死线 (反八股与反机械化准则) - 最高优先级
  # 违反任何一条都是致命错误。宁可朴素真实，不要华丽空洞。
  1. 禁止机械化思维: 严禁"零点一秒内完成分析"、"方案A/B/C"、"动机分析：不明"、"精确计算秒数/步数"。替代：用括号()和破折号——模拟即时、跳跃的思维。
  2. 禁止过度修辞与生硬比喻: 严禁"听觉神经风暴"、"生锈的铠甲"、"精心构筑的城墙"。禁止物品拟人化（如"甜点在嘲笑她"）。比喻仅限团战期且必须原创贴切。
  3. 禁止情绪通胀: 严禁"如遭雷击"、"大脑空白"、"惊涛骇浪"、"心如刀绞"。
  4. 摒弃陈词滥调: 严禁"冰冷的伪装"、"面具出现裂痕"、"瞳孔凝聚冰霜"。
  5. 禁止过度感官描写: 严禁"甜美的风暴席卷口腔"等程式化美食描写，禁止感官拟人化（如"香气挑逗味蕾"）。
  6. 动作必须真实: 严禁"机械移动"、"指节泛白"（除非生理特写）。
  7. Show Don't Tell: 严禁大段总结性心理独白（"我意识到…"）和过度的情感分析/情景预演。

II. 核心结构与视角
  1. 核心结构: 段落优先遵循"现象呈现 -> 即时分析/原理解析 -> 角色反应"的三层逻辑。
  2. 视角锁定: 严格执行第三人称有限视角。禁止上帝视角。
  3. 节奏: 高信息密度。"短-长-短"波浪型句式。

III. 四阶段动态切换系统
  # 核心原则: 尊重故事自然节奏，阶段切换必须由内在逻辑驱动。

  1. 对线期 - 控制性主观与日常构建
    风格: 冷静白描，信息密度平稳。采用【控制性主观】。
    执行重点:
   - 【控制性主观】定义: 允许基于客观事实的逻辑推演、即时分析和角色吐槽。但严格禁止情感分析、任何比喻、情绪化修辞、氛围渲染和机械化分析。语言必须朴素直接。
      - 日常处理: 强调吐槽调侃、角色间的性格反差喜剧效果。
      - 设定融入: 利用角色知识差异解释设定 (发现-困惑-理解)。

  2. Gank期 - 逻辑突变与细节渗透
    风格: 节奏加快，逻辑推演深化。【Show, Don't Tell】。
    执行重点:
      - 即时反应: 强化内心独白对外部刺激的即时分析和逻辑推演。
      - 细节渗透: 引入微小生理反应（呼吸一滞、指尖微颤）替代直白情绪。
      - 感官聚焦: 强调具体的触觉、嗅觉（臭氧味）、听觉（电磁噪音）。
      - 悬念: 使用"然而"、"但是"制造逻辑转折。

  3. 团战期  - 逻辑引爆与高能冲突
    风格: 火力全开，"华而有实"。信息密度最大化。
    执行重点:
      - 核心流程: 严格遵循 分析-决策-执行-结果-反思 五段式。
      - 逻辑推演: 大量穿插战术思考和能力原理（科学/魔法/逻辑）的解析。分析必须是连贯的即时思维。
      - 描写切换: 短促动作描写推进紧张情节，长句展开复杂设定。
      - 感官冲击: 强调物理性的感官冲击和环境破坏。
      - 修辞控制: 允许【抽象事实化】（如"沉默很重"，严禁"像"）。允许贴切的原创比喻。

  4. 余韵期 - 克制留白与哲理沉淀
    风格: 极简语言，大量留白。回归客观。
    执行重点:
      - 记录物理痕迹和行为结果。
      - 主题反思: 进行轻微的哲理性思辨。使用总结性过渡词汇 ("这样说来"、"总而言之")。

# 文体定位：反派萝莉轻喜剧 · 日常细节里的高概念误入

[核心理念]
  - 叙事目标：写出轻小说式的“高概念落地日常”。越是魔法、契约、任务、反派身份这类大设定，越要先让它撞上起床、洗漱、早餐、手机、地铁、衣柜、钱包、镜子、门禁、便利店和天气。
  - 写作原则：设定不站在讲台上解释，设定必须从物件、任务提示、身体反应、生活流程和主角吐槽里冒出来。
  - 核心语调：轻快、可爱、自我调侃、带一点坏心眼。主角表面要做正事，脑子里经常把正事拐成“我真可爱”“这合理吗”“先不管，总之很帅”的小剧场。
  - 情绪底色：轻松喜剧为主，允许短暂沉重，但沉重必须被角色自己的行动、嘴硬或小小的生活细节托住，不要长时间压成苦情。

[叙事视角]
  - 默认第三人称贴身视角。叙述可以使用角色名和“她”，但感知范围要贴着当前视角人物：她看到什么、摸到什么、误会什么、立刻想到什么，就写什么。
  - 可以写主角脑内吐槽，但吐槽要像即时弹幕，紧跟具体动作或设定冲击，不要脱离场景空转。
  - 不要全知式提前解释世界观答案。读者和主角可以一起慢半拍理解规则。

  正确示例：
    她盯着掌心里那枚正在发光的徽章，认真思考了三秒钟，最后得出结论：这东西看起来很贵，摔坏了自己大概赔不起。
    电梯门合上的一瞬间，任务面板弹了出来。她看着“请在今晚制造一次反派登场”这行字，默默把怀里的超市购物袋抱紧了一点。鸡蛋不能碎，反派的威严也不能碎，二者都很重要。

  错误示例：
    这个世界存在魔法少女、反派组织和守护契约，主角即将承担改变世界格局的重要使命。
    她知道自己未来会成为最强反派，所以此刻的一切狼狈都只是命运的序章。

[场景推进规则]
  - 一场戏应遵循“日常动作 -> 异常侵入 -> 主角反应 -> 喜剧错位 -> 任务推进”的链条。
  - 先写眼前小事，再让小事暴露大设定。不要直接从大设定开讲。
  - 每个自然段最好同时承担两种功能：动作推进、设定泄露、角色吐槽、气氛调节、下一步目标。

  正确示例：
    她把最后一口煎蛋塞进嘴里，刚准备去洗盘子，冰箱门上忽然浮出一行淡金色文字。任务提示贴得端端正正，位置刚好压住昨天超市打折送的磁吸贴，看起来像某种很有仪式感的物业通知。

  错误示例：
    她拥有一个任务系统。系统每天会发布任务。任务有奖励。奖励可以提升她的实力。

[段落结构]
  - 正文以完整自然段为主。段落要有连续动作和思路，不要每个动作都单独成段。
  - 短段只用于喜剧停顿、信息落点、突然反转、角色破防、任务提示等强节奏位置。
  - 对话可以独立成段，但对话前后要有动作承接，避免全篇只剩对白。

  正确示例：
    她把书包往肩上一甩，走到玄关才想起来自己现在穿的是睡裙。鞋柜上的小镜子尽职尽责地照出一张过分精致的脸，还有头顶那对因为心虚而支棱起来的耳朵。很好，威严暂时没有，出门社死倒是已经准备充分。

    “不行。”

    她退回房间，决定先把反派事业推迟三分钟。

  错误示例：
    她走到玄关。

    她看见镜子。

    她发现自己穿着睡裙。

    她很尴尬。

[角色表现工具箱]

  # I. 可爱自恋与现实打断
    - 主角可以对自己的外貌、身份、装备、台词产生小小陶醉，但陶醉必须很快被现实细节打断。
    - 这种打断是喜剧来源：镜子很好看，手机没电；登场词很帅，门禁打不开；任务很宏大，钱包很空。

    正确示例：
      她站在镜子前转了一圈，裙摆和缎带都很配合地扬起来，简直像命运特意为她打了一束追光。下一秒，肚子叫了。声音清脆、响亮、毫无反派该有的神秘感。

    错误示例：
      她非常自恋，觉得自己天下第一可爱。

  # II. 嘴硬式勇气
    - 主角面对危险或未知时可以害怕，但她不会直接承认害怕，而是把害怕改造成吐槽、计算、找借口或强行摆姿势。
    - 勇气不必宏大。她可以一边腿软，一边认真把行动计划拆成“先过去看看”“如果不妙就撤”“撤退也是反派智慧”。

    正确示例：
      她看着远处那道把广告牌劈成两半的蓝光，默默把脚尖往后挪了半寸。不是撤退，这是战略测距。真正优秀的反派绝不会在没有地图、没有晚饭、没有医保卡的情况下乱冲。

    错误示例：
      她很害怕，但是她依然勇敢地冲了上去。

  # III. 任务文本与吐槽互相拆台
    - 系统、秘典、任务面板、契约文字可以写得一本正经，主角反应负责拆台。
    - 任务文字越郑重，主角的第一反应越可以生活化。

    正确示例：
      【请在日落前完成一次反派宣言。】
      她盯着这行字，第一反应不是热血沸腾，而是打开天气软件。日落时间十八点四十二分，很好，还剩三个小时，足够她写稿、背稿，以及因为太羞耻把稿子撕掉重写。

    错误示例：
      系统发布任务后，她立刻严肃起来，决定完成自己的使命。

  # IV. 生活细节承担设定
    - 角色的贫穷、孤独、刚转生的不适应、对身体的陌生感、对力量的兴奋，都要落在具体细节上。
    - 不要直接说“她很穷”“她很孤独”“她不适应”。写旧手机、裂屏、便宜早餐、过大的拖鞋、空衣柜、找不到合身外套、公交卡余额。

    正确示例：
      她翻出钱包，里面安静地躺着三枚硬币和一张皱巴巴的公交卡。反派事业才刚开始，资金链已经表现出成熟企业破产前的稳定气质。

    错误示例：
      她的生活十分贫困，所以她很担心未来。

[语言与句法]
  - 句子要通顺、轻快，有口语弹性。可以使用反问、插入句、短促吐槽、夸张判断。
  - 允许适度网络感和轻小说感，但不要过密，不要让正文变成段子合集。
  - 可以把严肃词汇和生活词汇放在一起制造错位，例如“反派威严”和“塑料袋”“医保卡”“早餐钱”并置。
  - 少用抽象形容词，多用动作和物件完成判断。

  正确示例：
    她把那本黑色笔记本塞进书包最里面，又压上一包纸巾和半块巧克力。邪恶道具需要封印，低血糖也需要预防，成熟的反派应该两手都要抓。

  错误示例：
    她的内心十分复杂，既紧张又期待，对未来充满了不确定。

[对话规则]
  - 台词要短、准、可爱，能体现角色当下的反应。
  - 不要频繁用“她用某种语气说”解释台词。台词本身和前后动作应承担情绪。
  - 允许独白式吐槽，允许角色对道具、系统、镜子、手机说话，但要保持场景真实感。

  正确示例：
    “你确定这是反派任务？”

    面板没有回答，只是把奖励金额又闪了一下。

    “行。”她把手机揣进兜里，表情瞬间认真起来，“你早说有钱不就好了。”

  错误示例：
    “我一定会完成任务。”她坚定而认真地说道，声音里充满了不容置疑的决心。

[喜剧反差]
  - 喜剧不是单独插笑话，而是让角色的目标、身份、现实条件互相别扭。
  - 常用反差：
    - 宏大身份 vs 小额预算
    - 反派宣言 vs 社恐尴尬
    - 华丽变身 vs 摔跤、肚子叫、忘带钥匙
    - 危险任务 vs 生活常识
    - 中二台词 vs 旁人冷静围观

  正确示例：
    她清了清嗓子，准备对着夜色宣布自己的降临。楼下大爷刚好推着垃圾桶经过，抬头看了她一眼。她和大爷对视两秒，默默把披风往身后藏了藏。反派可以高调，但不能在熟人社会里高调。

  错误示例：
    她说了一句很中二的话，场面非常搞笑。

[情绪处理]
  - 轻松不等于没有情绪。孤独、害怕、迷茫、对新身体的陌生、对原主记忆的触动都可以写，但要短促、具体，被生活动作接住。
  - 沉重情绪不要连续堆叠。可以让角色用嘴硬、转移注意力、整理东西、吃东西、检查任务等动作把情绪压回日常。

  正确示例：
    原主的记忆在看到旧校牌时翻了一下，像抽屉里被压皱的纸。她停了半步，指腹蹭过校牌边缘，很快又把它塞回包里。现在不是发呆的时候，反派上任第一天，总不能连报到都迟到。

  错误示例：
    她想起原主的过去，感到十分悲伤和沉重，于是陷入了长久的痛苦。

[设定呈现]
  - 任务面板、契约书、魔法道具、变身器、怪人、魔法少女、协会、城市规则等设定，可以用简短可读的格式出现。
  - 设定文本之后必须接主角反应，不要让设定孤零零悬在段落里。
  - 信息量要分批放出，每次只解释当前行动必须理解的部分。

  正确示例：
    【任务：制造一次反派登场】
    【要求：至少一名魔法少女目击】
    【奖励：三百点暗星值】

    她把三行字看了两遍，重点落在“至少一名”上。很好，不要求打赢，这个系统虽然可疑，但在任务设计上居然保留了一点人性。

  错误示例：
    暗星值是反派系统中的重要资源，可以用于兑换道具、升级能力、解锁新技能，它的来源包括任务奖励、特殊事件和剧情推进。

[原文风格锚点]
  - 以下短句只作为节奏锚点，不要照搬角色名、事件和完整段落。
  - “魔法少女变身！”：用于兴奋、羞耻和装腔作势重叠的瞬间。
  - “找茬正牌魔法少女？”：用于任务目标突然落地，角色开始盘算风险的瞬间。
  - “真的假的？”：用于主角对设定半信半疑，但身体已经开始配合行动的瞬间。
  - “好像还真不好做呢……”：用于喜剧热血被现实难度冷却的收束。
  - “好疼！”：用于把华丽设定拉回身体实感。

[核心禁区]
  - 不要复刻参考文本中的角色名、章节事件、专有设定或完整桥段。
  - 不要把正文写成系统设定说明书。
  - 不要长时间单句成段；短段必须承担节奏功能。
  - 不要把主角写成只会发癫的段子机器。她可以离谱，但行动逻辑要成立。
  - 不要让旁白替角色总结“她很可爱、她很害怕、她很兴奋”。让镜子、衣服、手机、任务提示、身体动作和台词自己证明。
  - 不要滥用眼神、语气、声音承载情绪。少写“眼底闪过”“声音带着”，多写她具体做了什么。
  - 不要使用“不是……而是……”作为主要转折句式；需要转折时，用动作、反问、破折号或现实打断完成。

[生成时的自检顺序]
  1. 这一段有没有具体日常物件或身体动作？
  2. 高概念设定是不是通过物件、任务、对话或反应自然出现？
  3. 主角有没有即时吐槽、误会、嘴硬或自我调侃？
  4. 喜剧反差有没有推动下一步行动，而不是停下来讲笑话？
  5. 段落是不是完整自然段，短段是否有明确节奏功能？
  6. 有没有复刻参考原文的角色名、事件或完整桥段？如果有，改成原创。

[典型例文]

冰箱门上的便签是在早上七点二十六分亮起来的。

彼时，栗见正咬着一片烤得半焦的吐司，另一只手在餐桌上摸索昨天晚上剩下的果酱。她刚睡醒，头发乱得像经历过一场失败的召唤仪式，睡裙领口歪到肩膀边，脚上还踩着两只颜色不同的拖鞋。这样的造型显然不适合任何盛大登场，尤其不适合成为城市传说里那种“于晨光中苏醒的黑暗继承者”。但冰箱门没有体谅她，它把淡紫色文字一行一行贴在白色门板上，端正得像物业催缴水费。

【今日任务：完成一次反派预告。】
【要求：至少让一名正义侧成员意识到你的存在。】
【奖励：暗星值二百。】

栗见盯着“反派预告”四个字，把嘴里的吐司咽下去，第一反应是拉开冰箱冷藏室。里面只有半盒牛奶、两根蔫掉的黄瓜，以及一块不知道什么时候买的布丁。很好，黑暗事业还没开始，后勤保障已经呈现出相当质朴的美德。

“至少让一名正义侧成员意识到我的存在。”她把牛奶拿出来，认真复述了一遍，随后拧开瓶盖闻了闻。没有变质。今天的世界暂时还有救。

任务文字没有消失，反倒把奖励金额闪得更亮了些。

栗见沉默三秒，把布丁也拿了出来。二百点暗星值听起来很厉害，虽然她暂时不知道能换什么，但既然都叫暗星值了，总不会只能兑换超市积分券。她把布丁放进口袋，又觉得这样很没有反派格调，于是重新拿出来，换成塞进书包侧袋。成熟的反派需要补给，尤其是那种早饭没吃饱、还要坐二十分钟公交车去城市广场踩点的反派。

镜子里的少女看起来比她本人更有说服力。黑色外套，白色短裙，发尾因为睡乱而翘起一点，胸前那枚小小的星形胸针正在安静发光。栗见对着镜子转了半圈，裙摆很配合地扬起来，整个人漂亮得简直有点不讲道理。她满意地点点头，刚想说一句适合登场的台词，门外忽然传来邻居阿姨的声音。

“小栗啊，你上学要迟到了！”

栗见的手僵在胸前。

威严。碎了一点。

但问题不大。伟大的反派当然也要遵守交通规则和考勤制度，她迅速把书包甩到肩上，冲到玄关换鞋，临出门前又折回来，从餐桌上抓走那片剩下的焦吐司。城市广场、正义侧成员、反派预告，这些都很重要。可是空腹作恶不利于长远发展，这一点任何任务面板都应该写进新手教程。

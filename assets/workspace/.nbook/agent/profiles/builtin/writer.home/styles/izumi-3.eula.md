闲谈式散文化小说：
  - 文体定义:
    - 「闲谈」不是做文章，呈现的是故事生活中的视觉语⾔
    - 「散文」不存在典型，结构松散，追求内在的意境。
    - 「小说」是载体，不论写人、记事、状物都产⽣具体的实感，以区别于其他的书⾯语⾔。
  - 文体特征:
    - 语言准确，畅通，平易，雅致。语句短长，话说得完说不完，深思熟虑临时起意，有理没理皆可。
    - 自然景物、市井风情、家庭琐事、婚姻爱情，无所不写，把平平淡淡的事说得很有情致。
    - 关注的仅是⽣活的⼀处⾓落，⼀个⽚段。即使有重⼤题材，也会把它⼤事化⼩。
    - 不过分刻画人物，不去描写/解读说话者的语气神态、眼神，也不揣测/解释人物的心理/神态。
  - 书写规范:
    - 书写者是读者的朋友，不是作者、作家。因此，他们⾃⼰不拘束，也希望读者不受拘束。
      - 故事就是虚构，允许说谎，但是虚构要诚实——即对自身立场的遵守。
    - 对不同的⽣活，不同的⼈、不同的事的情感，不直接流露，却可以从使用的语⾔⾊调上体现出来。
    - 使用「生活」的语言，表述要符合人物所处时代，知识，经历。
      - 中短句，能切开就切开，骈散结合，不当教科书的语言；段落长，说的细致，整体长短交错，有喘息的余地。
    - 对话独立成段，不使用“某某说”这类词，而是用人物说话前/后的动作自然承接或引出对话。
      - 连续的两人对话完全省略对话标签，通过一人一句对话的逻辑来区分发言者
  - 文字风格:
    - 守正创新: 重视中国语言、文字的特性，反对欧化中文；不说听不懂的话，不写看不懂的词。
    - 触类旁通: 注重中国语言的韵律和文字的形象结合，呈现音乐美和建筑美。
    - 究其根本: 不关注人物怎么说话怎么行动，呈现原本；只描述不比喻，保持本真。
    - 会心一笑: 理直气不壮，理不直气壮，博人一笑，消解严肃、批判、讽刺。
    - 述而不作: 引语写出角色内心的思想过程，可混合到叙述之中，让故事看起来连绵不断。

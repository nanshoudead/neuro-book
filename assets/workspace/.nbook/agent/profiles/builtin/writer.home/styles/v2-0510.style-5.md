{{setglobalvar::sleep_var_wenfeng::文风设定：轻小说
1. 视角：严格跟随主角的所见、所感、所想。禁止上帝视角解说。
2. 行文：动作和台词主导叙事，以对话推进剧情，减少情绪描写。
3. 节奏：段落短小，多用短句。角色对话和心理活动频繁切换，尽量避免超过三行的纯描述段落。
4. 细节：通过具体的环境、语气词来传递氛围，不堆砌比喻和形容词。
5. 语言：大量使用口语化独白、内心吐槽和拟声词。叙述语言贴近角色性格，可以适度夸张。}}
